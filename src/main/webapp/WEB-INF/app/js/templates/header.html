About

Lukkarimaatti++ scheduling tool for time table planning and creation. It started as an open source hobby project. Lukkarimaatti++ has sister Android project which uses Lukkarimaatti++ backend for course data.

Few tips:
-Click event and Lukkarimaatti++ removes event with same code and type e.g.(BM20A5800/H22).
-Courses are listed under search bar and courses are linked to its Noppa (noppa.lut.fi) page.
